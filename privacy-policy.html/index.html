Politique de confidentialité — SenMoto

Dernière mise à jour : 5 juillet 2026

SenMoto ("l'Application") est une application de gestion destinée aux entreprises pour administrer leurs boutiques, publications et contenus. Cette politique décrit les données que nous collectons et comment elles sont utilisées.

1. Données collectées

  • Informations de compte : nom, adresse e-mail, rôle au sein de l'entreprise.
  • Contenu créé par l'utilisateur : publications, textes, images téléversées dans l'Application.
  • Localisation : utilisée uniquement, avec votre autorisation, pour situer votre boutique sur une carte.
  • Comptes de réseaux sociaux connectés (Facebook, TikTok) : lorsqu'un administrateur d'entreprise choisit de connecter un compte, nous stockons les jetons d'accès nécessaires pour publier du contenu en son nom.

2. Utilisation des données

Les données sont utilisées exclusivement pour faire fonctionner les fonctionnalités de l'Application : gestion des publications, affichage de la boutique sur une carte, et publication de contenu sur les plateformes tierces explicitement connectées par l'entreprise (Facebook, TikTok).

3. Partage avec des tiers

Nous ne vendons aucune donnée personnelle. Des données sont transmises aux plateformes tierces suivantes, uniquement lorsque l'entreprise a explicitement connecté son compte :

  • TikTok (API de publication de contenu) : pour publier le contenu choisi par l'entreprise sur son compte TikTok connecté.
  • Meta / Facebook : pour publier le contenu choisi par l'entreprise sur sa page Facebook connectée.

4. Conservation et suppression

Les jetons d'accès aux comptes tiers sont supprimés immédiatement lorsque l'entreprise déconnecte le compte concerné depuis l'Application. Les autres données sont conservées tant que le compte de l'entreprise est actif.

5. Sécurité

Les données sont stockées de manière sécurisée via notre fournisseur d'infrastructure (Supabase) avec des règles d'accès restreintes par entreprise.
